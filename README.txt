== Welcome to ROFL On Rails ==

ROFL On Rails is a Web Application Framework that was created to showcase a
preprocessed serverside implamentation of the LOLCODE programming language.

Features include ability to process HTTP GET parameters, and interface with MySQL

== Installation Instructions ==

1. Navigate to this directory in the Terminal and type 'make install' into the Terminal

== Creating a Project ==

1. Type 'CAN HAS FRAMEWORK' into the Terminal

== Creating a Website ==

1. Follow the instructions in the file 'README.txt', which will be created when you make a new project
